1. Administrateur (Profil Technique)

C'est celui qui prépare le terrain. Il n'intervient pas dans le contenu pédagogique.

    Rôle : Configurer l'instance de la faculté.

    Actions clés : * Lancer le Setup Wizard (Nom de la faculté, logo).

        Gérer les comptes utilisateurs (création, réinitialisation).

        Maintenir le référentiel des salles et des départements.

2. Coordinateur (Profil Métier / Power User)

C'est l'acteur central, souvent un chef de département ou de filière. C'est lui qui fait "vivre" l'application.

    Rôle : Orchestrer les sessions de soutenance.

    Actions clés :

        Importer les listes d'étudiants et d'enseignants (CSV/Excel).

        Définir les périodes de soutenance.

        Assigner les jurys et gérer les conflits via le moteur de contraintes.

        Valider et publier le planning final.

3. Enseignant / Membre du Jury (Profil Pédagogique)

L'utilisateur qui intervient le jour J pour l'évaluation.

    Rôle : Évaluer les travaux des étudiants.

    Actions clés :

        Consulter son planning personnel.

        Déclarer ses indisponibilités (si vous avez choisi cette option).

        Saisir les notes et les observations sur la fiche d'évaluation numérique.

4. Étudiant (Profil Utilisateur Final)

L'acteur qui bénéficie du service d'information.

    Rôle : Candidat à la soutenance.

    Actions clés :

        Vérifier les informations de son projet.

        Télécharger sa convocation officielle.

        Consulter son résultat final (Admis/Ajourné).
